Detailed Description

The main class for the simulation of a 2D world with many robots running around.

GraphicsArenaViewer or Tests call AdvanceTime to control the simulation and use the get*() functions to read out the current state of the simulation (i.e., the current positions and orientations of robots and obstacles).
